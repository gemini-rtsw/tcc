This file contains some notes on using the tcc; it is not a complete
user guide but describes some of the trickier aspects of the system.

Managing configurations.

To define a new configuration component; e.g. a new pointing origin:

1) Find the appropriate edit window.

2) Type a name for the new pointing origin into the box at the top of the
   window.

3) Type the X & Y values into the boxes marked X & Y.

4) Press the store button.

The name will then appear in the drop down lists of pointing origins and
be selected in the pointing origin component of the next level up in the
configuration heirachy (in this case in the target selector window).

To select an existing component, select it from the drop down list in either
the component edit window or the next level up.

To edit an existing component, select it, change the values on the edit
screen and press store.

To send a component to the telescope:

1) IF the  Apply button on the component edit window is disabled, enable
   it by clicking on the check button next to it.

2) Press the apply button.

Note that if you have edited the component definition but haven't saved it,
pressing apply will save the new definition and the old definition will 
be lost.

If you want to send a component to the telescope without defining a name for
it:

1) Select the blank line in the edit window drop-down list (or delete the
   name by double clicking on it and pressing the Delete key).

2) Enter the values you want in the data entry boxes.

3) Press apply.

Note that when you select the blank entry all the data entry fields are
cleared; if you find you have started entering values but have forgotten
to select the blank entry, delete the name rather than selecting the blank
entry to avoid loosing the data you have typed in.
