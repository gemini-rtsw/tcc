Significant changes since V1-4

1) The star selection function of the pointtest application has been
   integrated with the tcc; the "star selector" item on the "tools"
   menu brings up the "radar screen" display; selecting a star on this
   display copies the star position into the science target data entry
   screen. The telescope can then be slewed to this star using the normal
   tcc buttons.

   The pointtest application should now only be used for pointing tests.

2) The three handsets have been revised and they can now be aligned with
   RA/Dec, focal plane x/y (e.g. the acquisition camera), the instrument
   (i.e. relative to the instrument principal direction), Az/El or either
   of the peripheral wave-front sensors).

3) There is a (as yet incomplete) instrument origin management facility 
   attached to the instrument handset.

4) The pointing update tool is now started from the tools menu instead of
   being a separate application.

Significant changes since V1-5

1) The four target data entry screens and the star selector screen have been
   combined into one window and the star selector has been removed fom the 
   tools menu.

2) The three handsets have been combined into one window and the button for
   bringing up the handsets has been moved to the top-level toolbar window.

3) The TSD indicates the PCS status.
